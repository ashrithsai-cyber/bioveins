# BioVeins Lifesciences

A professional pharmaceutical website built with React, TypeScript, and Tailwind CSS.

## Features

- **Responsive Design**: Optimized for all devices
- **Product Showcase**: Interactive product galleries with detailed information
- **SEO Optimized**: Complete SEO infrastructure with meta tags, structured data, and sitemaps
- **Performance Optimized**: Fast loading with optimized assets
- **Accessibility**: WCAG compliant design
- **Modern UI**: Clean, professional interface using shadcn/ui components

## Tech Stack

- **Frontend**: React 18, TypeScript
- **Styling**: Tailwind CSS, shadcn/ui
- **Build Tool**: Vite
- **Routing**: React Router
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone <repository-url>
cd bioveins-lifesciences

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
src/
├── components/          # Reusable UI components
├── pages/              # Page components
├── assets/             # Static assets
├── lib/                # Utility functions
└── hooks/              # Custom React hooks
```

## SEO Features

- Meta tags for all pages

- JSON-LD structured data

The project is configured for deployment on any static hosting service like Vercel, Netlify, or traditional web servers.
